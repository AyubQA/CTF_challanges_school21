Задание 20: ФИНАЛЬНОЕ ЗАДАНИЕ - Найдите флаг!

Условия:
- Флаг находится в скрытой директории
- Флаг находится в архиве
- Флаг находится в бинарном файле
- Флаг закодирован в base64
- Флаг помечен строкой 'ENCODED_FLAG:'

Подсказка: Комбинируйте все изученные техники!
1. Найдите скрытую директорию
2. Распакуйте архив
3. Используйте strings для поиска в бинарном файле
4. Декодируйте base64
