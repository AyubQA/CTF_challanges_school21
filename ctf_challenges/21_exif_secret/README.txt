Задание 21: Секрет в EXIF

Категория: Forensics (метаданные)
Сложность: Medium

В изображении photo.jpg скрыт флаг в полях EXIF. Нужно извлечь его с помощью exiftool.

Подсказка: Используйте команду exiftool photo.jpg для просмотра всех метаданных.
Также можно попробовать: strings photo.jpg | grep school21
