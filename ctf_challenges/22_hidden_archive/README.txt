Задание 22: Спрятанный архив

Категория: Forensics / Stegano
Сложность: Medium

В файле picture.png с помощью LSB-стеганографии спрятан ZIP-архив. Нужно извлечь архив, распаковать его и найти флаг.

Подсказка: Используйте инструменты для LSB стеганографии:
- zsteg picture.png
- steghide extract -sf picture.png
- stegsolve (GUI инструмент)
- Или используйте Python скрипт для извлечения LSB

После извлечения архива распакуйте его и найдите флаг.
