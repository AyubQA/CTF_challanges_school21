Задание 23: Потерянные фрагменты

Категория: Forensics (файловая система)
Сложность: Hard

Дан образ диска disk.img. Нужно восстановить удалённый файл, содержащий флаг, с помощью foremost или scalpel.

Подсказка: 
1. Смонтируйте образ: sudo mount -o loop disk.img /mnt
2. Или используйте foremost: foremost -i disk.img -o output/
3. Или scalpel: scalpel -c /etc/scalpel/scalpel.conf disk.img
4. Проверьте восстановленные файлы на наличие флага

Примечание: В локальной версии файл может быть просто удален, но данные остались на диске.
