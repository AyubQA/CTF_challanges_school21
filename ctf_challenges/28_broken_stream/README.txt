Задание 28: Сломанный поток

Категория: Crypto (AES)
Сложность: Hard

В скрипте на Python реализовано шифрование AES в режиме CBC, но есть уязвимость: статический IV. Нужно восстановить флаг из зашифрованного текста.

Подсказка:
1. Изучите скрипт encrypt.py - найдите статический IV
2. Проблема: если IV одинаковый для всех сообщений, можно использовать атаку
3. В режиме CBC: plaintext = decrypt(ciphertext) XOR previous_ciphertext (или IV для первого блока)
4. Если известен один блок открытого текста, можно восстановить другие
5. Или используйте готовый скрипт для атаки на слабый IV

Команды:
- python3 decrypt.py encrypted.bin
- Или напишите свой скрипт для расшифровки
