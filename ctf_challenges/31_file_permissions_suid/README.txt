Задание 31: Права доступа

Категория: Linux
Сложность: Medium

Флаг находится в файле /root/flag.txt, но текущий пользователь не имеет к нему доступа. Нужно найти SUID-бинарник, который позволит прочитать файл.

Подсказка:
1. Найдите SUID файлы: find / -perm -4000 2>/dev/null
2. Или: find /usr/bin /bin -perm -4000
3. Проверьте каждый SUID бинарник на возможность чтения файлов
4. Некоторые команды могут читать файлы: cat, less, more, head, tail
5. Если найдете SUID версию cat: ./suid_cat /root/flag.txt
6. Или используйте strings на бинарнике: strings suid_binary | grep school21

Примечание: В локальной версии SUID бинарник может быть в текущей директории.
