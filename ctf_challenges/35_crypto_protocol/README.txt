Задание 35: Крипто-протокол

Категория: Crypto (Diffie-Hellman)
Сложность: Hard

Даны перехваченные параметры обмена Диффи-Хеллмана. Нужно вычислить общий секрет и расшифровать флаг.

Подсказка:
1. Изучите файл с параметрами: p, g, A, B (публичные значения)
2. Если модуль p маленький, можно вычислить дискретный логарифм
3. Используйте SageMath или Python для вычислений
4. Общий секрет: K = A^b mod p = B^a mod p
5. Используйте K для расшифровки флага (обычно как ключ для AES)

Пример Python:
```python
from Crypto.Util.number import *
p = ... # модуль
g = ... # генератор
A = ... # публичное значение Алисы
B = ... # публичное значение Боба
# Вычислите a или b (дискретный логарифм)
# K = pow(A, b, p) или pow(B, a, p)
```
