Задание 38: Шифрование XOR

Категория: Crypto (XOR)
Сложность: Medium

Даны зашифрованный текст и часть ключа. Нужно восстановить полный ключ и расшифровать флаг.

Подсказка:
1. XOR обратима: plaintext XOR key = ciphertext, значит ciphertext XOR key = plaintext
2. Если известна часть ключа, попробуйте перебрать остальное
3. Если известна часть открытого текста (например, "school21{"), можно вычислить часть ключа
4. Напишите Python скрипт:
```python
ciphertext = bytes.fromhex('...')
known_key = b'...'
# Попробуйте расшифровать с известным ключом
# Или используйте известный plaintext для вычисления ключа
```

Команды:
- python3 xor_decrypt.py encrypted.bin partial_key.txt
- Или используйте онлайн калькулятор XOR
